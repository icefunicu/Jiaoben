# AI 编辑器终极对比 (AI Editor Matrix)

一个现代化、响应式的 Web 工具，用于全方位对比当前主流的 AI 代码编辑器（如 Cursor, Windsurf, Copilot, Trae 等）。

## 🎯 项目目标

为开发者提供一个清晰、实时更新的决策参考，帮助选择最适合自己的 AI 辅助编程工具。

## ✨ 功能特性

- **多维度对比**: 涵盖价格、免费额度、支持模型 (Claude 3.5, GPT-4o 等)、上下文窗口等关键指标。
- **实时搜索**: 快速筛选特定的编辑器。
- **分类过滤**: 支持按 "免费" 或 "付费" 模式快速筛选。
- **视觉体验**: 采用 UI/UX Pro Max 设计流，拥有流畅的动画和暗色模式界面。

## 🚀 如何使用

本项目是一个纯静态 Web 应用 (HTML/CSS/JS)，无需复杂的后端部署。

1. **直接运行**:
   在浏览器中直接打开 `index.html` 文件即可使用。

2. **本地开发**:
   如果需要修改样式或逻辑，建议使用 VS Code + Live Server 插件启动，以获得实时预览效果。

## 📂 文件结构

- `index.html`: 页面结构
- `style.css`: 样式定义 (包含设计系统变量)
- `script.js`: 数据渲染与交互逻辑
- `data.json`: 编辑器数据源 (如有)

## 🤝 贡献

如果您发现编辑器数据有更新或遗漏，欢迎提交 PR 修改数据源!
