# 沉浸式阅读器 (Immersive Reader)

模仿 Kindle 或 E-ink 效果，一键去除网页上所有干扰元素，甚至能将当前页面转换成“报纸排版”风格。

## 功能特性
- **一键纯净**: 智能提取网页正文，去除广告和侧边栏。
- **报纸排版**: 支持多栏布局，模仿传统报纸阅读体验。
- **电子墨水屏风格**: 高对比度黑白模式，保护视力。

## 🎨 预期效果

- **📖 悬浮按钮**: 页面右下角会出现一个书本图标（📖）。
- **纯净阅读视图**: 激活后，页面中心会弹出一个覆盖全屏的阅读器，仅显示文章标题和正文，背景为舒适的羊皮纸色。

## 🛠️ 使用说明

### 1. 激活/退出
有三种方式可以切换阅读模式：
- **点击按钮**: 点击页面右下角的悬浮按钮（📖）。
- **快捷键**: 按下 `Alt + R`。
- **菜单命令**: 通过油猴插件菜单选择 "切换阅读模式"。

### 2. 控制面板
在阅读器右上角有一个控制面板，可以：
- **切换主题**: 羊皮纸 (默认)、墨水屏 (E-ink)、夜间模式。
- **切换布局**: 经典模式 (单栏)、报纸模式 (双栏)。
- **朗读 (TTS)**: 点击耳机图标 (🎧) 或按 `Alt + P` 开始/暂停朗读全文。

## 安装
请将 `immersive-reader.user.js` 的内容复制到 Tampermonkey 或其他用户脚本管理器中。

## 开发
本项目使用标准 Userscript 结构。
核心依赖: [Readability.js](https://github.com/mozilla/readability) (通过 CDN 引入)
